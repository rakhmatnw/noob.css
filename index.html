Introduction

This is a simple, not fancy at all, mini CSS framework. A noob framework created by a noob. I made this little framework not to compete with other frameworks out there. This project act as a medium for me to learn. Because I prefer to learn by building something.

The content & components inside this framework is still limited, but I'll keep updating it to add new components in the future. If you want to try it, here is how:
