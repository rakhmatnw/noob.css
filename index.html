Introduction

This is a simple, not fancy at all, mini CSS framework.
I made this little framework not to compete with other frameworks out there.
This project is just a medium for me to learn and be comfortable with SASS and CSS best practices .

How To Use

I couldn't guarantee you that this frameworks is the best but you are absolutely welcome to try and use it in your project if you want to.

  1. Download kecik.css file here
  2. Link it to your project
